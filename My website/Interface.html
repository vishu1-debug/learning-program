INTERFACE

The process of specitying/representing ful abstract form/generalise form is called interface. Interface is the way/form through which java represents the concept of fully abstract class.

With the help of interface, java achieves multiple inheritance. In java, interface keyword is used to create an interface. We can specify instance variable and method/function prototype within interface as member.

The interface variable/data member of instance is by default public,static and final,while the method prototype within the interface is considered as abstract and public defaultly.

NOTE (1):-An interface can be utilised/implementes try any number of calsses and a single class can also utilised/implement multiple (more than one) interfaces.

NOTE (2):-In java, implements keyword is used to utilise/implement the interface.

Syntex to create an Interface :- 

    
        
  • interface interface-name
            {
                interface variable/field-1;
                interface variable/field-2;
                --------------------------;
                --------------------------;
                interface variable/field-n;
                method prototype-1;
                method prototype-2;
                ------------------;
                ------------------;
                method prototype-n;
            }

NOTE :-By default access-specifir for interface is public ,ehile the class is friendly/defaultiy.

An interface can be utilised/implementes by one or more classes using implements keyword.The class who implements the interface must define the body of all abstract methods specified within interface. If a class performs partial utilisation of abstract method , themn that class must be declared as abstract otherwise compiler produce an error.

NOTE :-Like abstract class we can't create the object of an interface.However,we can create the reference of an interface.

    Partial utilisation of abstract method of interface :-

    As we know, a class utilises/implements all abstract method of an interface. But in java, it is also provision to utilise abstract method according to need for this, we have to proceed the class with abstract keyword which implement an interface.

    1. Write a program to partial utilisation of abstract method of interface. 
              //partial utilisation of abstract method of interface.
        interface Ddemo
        {
            void display();
            void show();
            void disp();
        }
        abstract class Xayz implements Ddemo
        {
            public void display()
            {
                System.out.println("Hello Vishu");
            }
            public void show()
            {
                System.out.println("Hiii Vishu");
            }
        }
        class AbstractMethod extends Xayz
        {
            public void disp()
            {
                System.out.println("Bye Bye Vishu");
            }
            public static void main(String []args)
            {
                AbstractMethod aa=new AbstractMethod();
                aa.display();
                aa.show();
                aa.disp();
            }
        }
